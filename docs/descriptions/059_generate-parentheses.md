# 059. 括号生成

- **题目链接**：[括号生成](https://leetcode.cn/problems/generate-parentheses/description/?envType=study-plan-v2&envId=top-100-liked)
- **题目标签**：见 [热题100-学习计划顺序.md](../热题100-学习计划顺序.md)


## 题目描述
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

示例 1：

输入： n = 3

输出： ["((()))","(()())","(())()","()(())","()()()"]

示例 2：

输入： n = 1

输出： ["()"]

提示： 1 <= n <= 8
